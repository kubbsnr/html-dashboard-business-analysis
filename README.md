# HTML Dashboard – Business Analysis Project

Bu proje, iş analizi bakış açısıyla hazırlanmış basit bir web tabanlı dashboard çalışmasıdır.

## Amaç
İş verilerinin daha anlaşılır hale getirilmesi ve temel KPI’ların görselleştirilmesi amacıyla oluşturulmuştur.

## Kapsam
- Temel metriklerin görsel sunumu  
- Basit ve kullanıcı dostu dashboard tasarımı  
- İş analizi ve raporlama odaklı yaklaşım  

## Kullanılan Teknolojiler
- HTML  
- CSS  
- JavaScript  

## Açıklama
Bu proje yazılım geliştirme odaklı değil, iş analizi ve veri görselleştirme perspektifiyle hazırlanmıştır.
